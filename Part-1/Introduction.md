# Introduction to Theory of Computation & Fundamental Concepts

## What is Theory of Computation?

Theory of Computation (TOC) is a branch of Computer Science that studies:

- What problems can be solved by computers.
- How efficiently they can be solved.
- Which problems can never be solved by any computer.

It provides the mathematical foundation of Computer Science.

## Definition of Computation

Computation means the process of solving a problem by following a sequence of well-defined instructions (Algorithm).

In simple words:

**Computation = Input → Processing → Output**

### Example

```
Input: 5, 3
Algorithm: Add two numbers
Output: 8
```

## Algorithm

An algorithm is a finite sequence of well-defined instructions used to solve a problem.

### Characteristics

- Finite
- Definite
- Effective
- Input
- Output

### Example

```
Algorithm Add(A, B)

Step 1: Read A, B
Step 2: Sum = A + B
Step 3: Print Sum
```

## Why Study TOC?

TOC helps us:

- Design compilers
- Design programming languages
- Build search engines
- Artificial Intelligence
- Cryptography
- Pattern Matching
- Operating Systems
- Software Verification
