# Hierarchy of Problems

Problems are divided into two major categories:

```
                                  Problems
                                      |
                          -------------------------
                          |                       |
                      Solvable              Unsolvable
                          |
                    ----------------------
                    |                    |
                Tractable          Intractable
```

## Solvable Problems

Problems that have an algorithm. The computer can solve them.

**Examples:**
- Sorting
- Searching
- Addition
- Matrix Multiplication

### Solvable Problems Categories

Solvable Problems are divided into two types:

#### Tractable Problems

These can be solved efficiently, usually in polynomial time.

**Examples:**
- Binary Search: O(log n)
- Merge Sort: O(n log n)
- BFS: O(V+E)
- DFS: O(V+E)

#### Intractable Problems

Problems that are solvable but require enormous time, usually exponential or factorial complexity.

**Examples:**
- Travelling Salesman Problem (exact)
- Knapsack
- Sudoku Solver
- Chess

## Unsolvable Problems

Problems for which no algorithm exists. No computer can solve them.

**Example:**
- Halting Problem
